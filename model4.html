Cross Attention Model: Conditional Generation with Cross Attention

This model was trained using the chunked dataset with both vocal and instrumental tracks. It leverages cross-attention to better integrate information between the different track classes. By using cross-attention, the model can generate specific instrumental tracks conditioned on the vocal input, improving the quality and coherence of the generated audio.

Unlike previous models, which were trained independently for each track, this model generates multiple tracks in a unified manner. The cross-attention mechanism enables the model to use the vocal track as context while generating the instrumental parts, providing a more comprehensive and context-aware generation process.
